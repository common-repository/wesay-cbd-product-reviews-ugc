== Installation ==

= Minimum Requirements =

* WordPress 3.5.1 or greater
* PHP version 5.2.0 or greater
* WooCommerce 3.0 or greater

1. Install WeSay reviews plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. After activating WeSay, click on the WeSay link in the left hand side bar of your WordPress admin.
3. You will now need to register an account with WeSay or log in with your existing account.
4. Copy the key from WeSay console and customize the different settings on the WeSay Settings page and click update when done.
